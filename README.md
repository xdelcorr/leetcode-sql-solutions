# LeetCode SQL Solutions

This repository contains LeetCode SQL solutions organized by difficulty and problem.

Each problem lives in its own folder and includes:

- A `.md` writeup with the problem summary, approach, key idea, explanation, and notes
- A matching `.sql` file with the final query only

```text
└── easy/

    │   ├── 1148-01-article-views.md

        ├── 619-biggest-single-number.md

```

## Solution Format

The Markdown writeups generally follow this structure:

- `Problem`
- `Approach`
- `Key Idea`
- `Step-by-Step Explanation`
- `SQL Query / Code`
- `Notes`

The SQL files are kept separate so the final answer can be reviewed quickly without the surrounding explanation.

## Current Coverage

- `easy`: 5 problems

More SQL problems and additional difficulty levels can be added as the collection grows.

## Goal

The goal of this repo is to keep SQL answers easy to read, easy to review, and easy to revisit later.
